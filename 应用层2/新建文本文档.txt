  数据接入方式

  方式一：API 接口

  在"数据管理 → API接口"中配置，每个数据集独立一个端点：

  ┌──────────────────────┬─────────────────────────────────────────────────────────────────────┐
  │        配置项        │                                说明                                 │
  ├──────────────────────┼─────────────────────────────────────────────────────────────────────┤
  │ API URL              │ 接口地址，支持 GET/POST                                             │
  ├──────────────────────┼─────────────────────────────────────────────────────────────────────┤
  │ Authorization        │ Bearer Token 或其他认证头                                           │
  ├──────────────────────┼─────────────────────────────────────────────────────────────────────┤
  │ 自定义 Headers       │ JSON格式附加请求头                                                  │
  ├──────────────────────┼─────────────────────────────────────────────────────────────────────┤
  │ 数据路径（JSONPath） │ 从响应中提取数组，如 data.list 或 result.rows（留空则直接用根数据） │
  ├──────────────────────┼─────────────────────────────────────────────────────────────────────┤
  │ 自动刷新间隔         │ 5分钟/15分钟/1小时，自动后台拉取                                    │
  ├──────────────────────┼─────────────────────────────────────────────────────────────────────┤
  │ 测试连接             │ 保存前可先测试并预览响应                                            │
  └──────────────────────┴─────────────────────────────────────────────────────────────────────┘

  方式二：文件导入（CSV / Excel / JSON）

  1. 在"数据管理 → 文件导入"中选择目标数据集
  2. 拖拽或点击上传文件
  3. 系统自动识别列名并提供字段映射界面（你的列名可以和模板不同，手动对应即可）
  4. 确认导入后数据持久化到 localStorage

  下载数据模板

  页面下方"下载数据模板"区域可为每个数据集下载 CSV 模板，包含字段名和示例行，对照填入真实数据后直接导入。
